# QUESTÃO 3

- Nessa questão, nosso objetivo é desenvolver a função xbyte, que 
será passado como argumentos um numero unsigned em um numero signed, 
mantendo o sinal, e o byte que ela quer extrair. Para isso basta 
usar manipulação de bits, a ideia é que a gente jogue o byte desejado 
para o topo(byte 3), para descartar a parte esquerda, armazenando esse 
valor em uma variavel inteira para que quando executemos o proximo 
passo (shift logico para a direita) ela conserve o sinal, e depois 
jogue ele para o byte 0. Então nossa função fica: 

    int xbyte (packed_t word, int bytenum) {
        int result = word << ((3 - bytenum) * 8);

        return (result >> 24);
    }
